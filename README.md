# SarıDefter — Kişisel Öğrenme Günlüğü

Yazılımcılar ve sürekli öğrenenler için geliştirilmiş, **retro / neo-brutalist** tasarıma sahip minimal bir çalışma notu (flashcard) uygulamasıdır.

## Özellikler
- 📝 **Akış Modu**: Tuttuğunuz kod parçacıklarını, dilleri (C#, React vs.) ve aklınıza gelen kavramları liste halinde saklayın.
- 🎴 **Tekrar Modu**: Günlük tekrar metodu (Flashcard) mantığı ile rastgele sıralanan notlarınızı okuyarak öğrenmeyi kalıcı hale getirin.
- 🌓 **Aydınlık/Karanlık Tema**: Göz yormayan gri-siyah veya yüksek kontrastlı göz alıcı sarı-beyaz retro tema.
- 💾 **Yerel Kayıt**: Verileriniz sunucuda değil, doğrudan kendi tarayıcınızda (Local Storage) güvende tutulur. İnternet olmadan da çalışır!

## Kurulum ve Çalıştırma

Projeyi bilgisayarınızda çalıştırmak için aşağıdaki komutları sırasıyla terminalinizde çalıştırın:

```bash
# Bağımlılıkları yükleyin
npm install

# Geliştirici sunucusunu başlatın
npm run dev
```

Uygulama otomatik olarak `http://localhost:5173` üzerinde çalışmaya başlayacaktır.

## Teknoloji Yığıtı (Tech Stack)
- **React 18** (Fonksiyonel Bileşenler & Hooks)
- **TypeScript** (Güvenli Tip Kontrolü)
- **Vite** (Hızlı derleme ve sıcak modül değişimi)
- **Lucide React** (Minimalist SVG İkon Seti)
- **CSS Modules** (Kapsüllenmiş ve modern stiller)
